🚀 Quick Start
Prerequisites
Node.js 16+

npm or yarn

Installation
Clone the repository

bash
git clone https://github.com/sparrow-pixeleye/apratim-s-ai-2.0.git
cd apratim-s-ai-2.0
Install dependencies

bash
npm install
Start development server

bash
npm run dev
Open your browser

text
http://localhost:3000
Building for Production
bash
npm run build
Deployment
bash
npm run deploy
